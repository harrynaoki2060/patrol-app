# GAS デプロイ手順

## 1. Google スプレッドシートを作成

1. https://sheets.google.com を開く
2. 「空白のスプレッドシート」を新規作成
3. タイトルを「安全パトロール」に変更

## 2. Apps Script エディタを開く

1. スプレッドシートのメニュー →「拡張機能」→「Apps Script」
2. エディタが開いたら、既存の `function myFunction() {}` を全削除
3. `Code.gs` の内容を全てコピー＆ペースト
4. 💾 保存（Ctrl+S）

## 3. API キーを登録する

1. Apps Script エディタの関数選択欄で `setupApiKey` を選択
2. `setupApiKey()` 内の `patrol-2026-change-me` を任意のキーに書き換え
   例: `patrol-2026-abc123`
3. 「実行」ボタンをクリック
4. 初回は「承認が必要です」ダイアログが出るので「権限を確認」→承認

## 4. Web App としてデプロイ

1. 右上「デプロイ」→「新しいデプロイ」
2. 歯車アイコン → 「ウェブアプリ」を選択
3. 以下の設定にする:
   - 説明: `安全パトロール v1`
   - 次のユーザーとして実行: **「自分」**
   - アクセスできるユーザー: **「全員」**
4. 「デプロイ」をクリック
5. 表示された **ウェブアプリURL** をコピーして保存
   例: `https://script.google.com/macros/s/AKfycb.../exec`

## 5. 動作確認

ブラウザで以下の URL を開いて `{"ok":true,...}` が返れば成功:

```
https://script.google.com/macros/s/【あなたのID】/exec?action=health&key=【APIキー】
```

## 6. アプリに設定を入力

アプリの「設定」画面（歯車アイコン）に戻り:
- **GAS Web App URL**: 手順4でコピーしたURL
- **API キー**: 手順3で設定したキー

を入力して「保存」。

---

## コードを変更した場合の再デプロイ

「デプロイ」→「デプロイを管理」→ 鉛筆アイコン →「バージョン」を「新バージョン」にして「デプロイ」
