# GAS デプロイ手順

## 前提

- Google スプレッドシートを1つ用意しておく（既存でも新規でも可）

---

## 手順

### 1. スプレッドシートを作成

1. https://sheets.google.com を開く
2. 新規作成 → タイトルを「安全パトロール」にする
3. URL から **スプレッドシートID** をコピーして控えておく
   ```
   https://docs.google.com/spreadsheets/d/【ここがID】/edit
   ```

---

### 2. Apps Script を新規作成（スタンドアロン方式）

1. https://script.google.com を開く
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を「安全パトロールAPI」に変更
4. 左ペインの `コード.gs` を選択
5. 中身を全て削除 → **Code.gs の内容を全てコピー＆ペースト**
6. 保存（Ctrl+S）

> **コピー前に確認**: ファイルの先頭行が
> `// Google Apps Script Web App` になっていること

---

### 3. スプレッドシートIDを登録（手動実行）

1. エディタ上部の関数プルダウンで **`setupSpreadsheet`** を選択
2. `setupSpreadsheet()` 内の `YOUR_SPREADSHEET_ID_HERE` を手順1でコピーしたIDに書き換え
3. 「実行」ボタンをクリック
4. 「承認が必要です」ダイアログ → 「権限を確認」→ 自分のアカウントで承認

---

### 4. APIキーを登録（手動実行）

1. 関数プルダウンで **`setupApiKey`** を選択
2. `patrol-2026-change-me` を自分のキーに書き換え（例: `patrol-abc-2026`）
3. 「実行」ボタンをクリック（承認済みなのでそのまま実行される）

---

### 5. テスト実行（確認）

1. 関数プルダウンで **`testHealth`** を選択
2. 「実行」→ 下部の「実行ログ」に以下が表示されれば OK:
   ```json
   {"ok":true,"spreadsheetId":"...","timestamp":"2026-07-23 ..."}
   ```

---

### 6. Web App としてデプロイ

1. 右上「デプロイ」→「新しいデプロイ」
2. 歯車アイコン → **「ウェブアプリ」** を選択
3. 設定:
   - 次のユーザーとして実行: **「自分（自分のメールアドレス）」**
   - アクセスできるユーザー: **「全員」**
4. 「デプロイ」をクリック
5. 表示された **ウェブアプリURL** をコピーして保存
   ```
   https://script.google.com/macros/s/AKfycb.../exec
   ```

---

### 7. ブラウザで動作確認

以下のURLをブラウザで開く（URL内のIDとキーは自分のものに置き換え）:

```
https://script.google.com/macros/s/【デプロイID】/exec?action=health&key=【APIキー】
```

レスポンスが以下ならば成功:

```json
{"ok":true,"spreadsheetId":"...","timestamp":"2026-07-23 14:30:00"}
```

---

## コードを変更した場合の再デプロイ

「デプロイ」→「デプロイを管理」→ 鉛筆アイコン →「バージョン」を「新バージョン」→「デプロイ」

---

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| `{"error":"Forbidden","code":403}` | キーが違う。`?key=` を確認 |
| `{"error":"...","code":500}` | スプレッドシートIDが未設定。`setupSpreadsheet` を再実行 |
| `{"error":"Unknown action","code":400}` | `?action=health` のスペルを確認 |
| HTMLが返ってくる（ログイン画面） | デプロイの「アクセス」が「全員」になっているか確認 |
